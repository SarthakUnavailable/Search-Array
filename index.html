Search-array

Searches an element in an array

Download this project as a .zip file Download this project as a tar.gz file

INTRODUCTION

This program aims to search for an element in an array and displays a suitable message based on the existence of the element in the array.

STEPS INVOLVED

  1. User inputs elements
  2. Program sorts the array using Bubble Sort and displays the sorted array
  3. User inputs the element to be searched for
  4. Program searches for the element using Binary Search and displays a suitable message
